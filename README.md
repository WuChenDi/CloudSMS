# Cloud SMS

Cloud SMS 是我在2017年开发的第一个前端项目，标志着我前端开发之旅的起点。该项目专注于解决早期浏览器的兼容性问题，为传统企业环境提供可靠的短信服务界面。

## 特性

- **IE兼容性**：目前支持IE7及以上浏览器，但在IE9以下的版本中仍存在圆角样式无法显示的问题，后期会逐步优化。
- **项目地址**：<https://mccll.pages.dev/>

## 开发历程

- **2017年**：项目初始开发，作为前端学习的第一个完整项目
- **2025-11-06**：文档更新，项目重新部署至 Cloudflare Pages

*注：该项目具有历史纪念意义，代码风格不作调整，保留了早期初始的开发习惯。*
